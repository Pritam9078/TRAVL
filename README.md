# Travl - A Kaleidoscopic Journey Through India

A minimal, responsive travelogue webpage created for English Project (Grade XI).

## Project Overview

**App Name:** Travl  
**Theme:** "The Kaleidoscopic View – A Travelogue"  
**Academic Year:** 2025-26

## Features

- **Clean & Minimal Design:** Soft neutral color palette with readable typography
- **Fully Responsive:** Adapts smoothly between desktop and mobile views
- **No External Dependencies:** Built with pure HTML and CSS only
- **Print-Friendly:** Includes print styles for academic submission

## Structure

### Sections Included:
1. **Header Section** - App name, title, and subtitle
2. **Introduction Section** - Overview of travel experiences
3. **States Section** - Four destination cards:
   - Odisha
   - Kerala
   - Shillong (Meghalaya)
   - Goa
4. **Reflections Section** - Personal insights and learnings
5. **Footer Section** - Project attribution

### Each State Card Contains:
- State name as heading
- 2-3 lines of narrative account
- 1-2 lines of culture and history
- 1 line highlighting local flavour

## Technical Details

- **HTML5** with semantic structure
- **CSS3** with Flexbox/Grid for responsive layout
- **Mobile-First** responsive design
- **Cross-Browser Compatible**
- **Accessibility Focused** with proper heading hierarchy

- **Desktop:** 1200px and above

## Color Palette

- **Primary:** #2c3e50 (Dark Blue-Gray)
- **Secondary:** #3498db (Blue)
- **Accent:** #e74c3c (Red)

## Usage

1. Open `index.html` in any modern web browser
2. The page will automatically adjust to your screen size
3. All content is self-contained in the two files
